# Deep Learning Repository

This repository contains various deep learning projects and implementations with a focus on Generative Adversarial Networks (GANs). The main focus is the **WGAN-GP** (Wasserstein GAN with Gradient Penalty), presented in the notebook `notebooks/WGAN_GP.ipynb`. The repository includes relevant resources such as model architectures and reports related to the project.

## Repository Structure

- `notebooks/WGAN_GP.ipynb`: This Jupyter notebook implements a Wasserstein GAN with Gradient Penalty. The GAN is trained to generate data samples, with the gradient penalty technique improving the stability of training.
  
- `discriminator_model.png`: Diagram showing the architecture of the discriminator used in the WGAN-GP.
  
- `generator_model.png`: Diagram showing the architecture of the generator used in the WGAN-GP.
  
- `presentazione-deep-learning.pdf`: A presentation providing an overview of deep learning principles, applications, and the WGAN-GP implementation.
  
- `relazione-deep-learning.pdf`: A report discussing the deep learning project, including the theory behind WGAN-GP and the results of training.
  
- `training_epochs.pdf`: A chart showing the training progress over epochs during the training of the WGAN-GP model.

## Project Overview
The WGAN-GP is an advanced version of the traditional GAN, designed to improve the convergence of GANs by using the Wasserstein distance as the loss function and incorporating a gradient penalty. This approach helps to avoid problems such as mode collapse and provides better stability during training.
